开发环境

在搭建过程中，可以使用 npm 构建依赖。因此首先需要 node.js，直接下载安装即可。

新建项目

新建一个项目，结构如下：

app: 开发目录

  • components 用来存放组件
    • Hello.js 第一个组件文件
  • main.js 整个项目的入口文件

build: 打包后的目录

  • index.html 是最终要展示的页面
  • bundle.js：打包后的js文件
